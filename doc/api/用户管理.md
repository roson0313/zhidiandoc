# 用户管理

## 获取手机号
``` js
mid: 3,
data: {
	tel_password: params.session_key, //解密手机号密码
	tel_newiv: params.iv, //解密手机号初始向量iv
	tel_encrydata: params.encryptedData //手机号原始密文
}
```
